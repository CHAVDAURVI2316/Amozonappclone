# 🛒 Amazon Clone – Frontend E-Commerce Website

## 📌 Project Overview
This project is a frontend clone of the Amazon e-commerce website, developed to replicate the core layout and user interface of a real-world online shopping platform.

The goal of this project is to demonstrate strong frontend development skills, including responsive design, UI structuring, and interactive components using modern web technologies.

---

## 🚀 Features

- 🏠 Amazon-style homepage layout  
- 🔍 Search bar UI  
- 🛍️ Product listing sections  
- 📦 Category-based browsing  
- 🧾 Product cards with price & details  
- 🛒 Add to Cart button (UI level)  
- 📱 Responsive design for mobile & desktop  
- 🔝 Sticky header & navigation bar  
- 🎯 Clean and user-friendly interface  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript
